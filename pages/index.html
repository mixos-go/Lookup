Masalahnya

Jualan di dua platform, kerjanya dobel.

Cara manual

  • Buka Shopee Seller Center, edit stok dan harga produk satu-satu.
  • Pindah ke TikTok Shop Seller Center, ulangi semua langkah yang sama.
  • Kelupaan update salah satu toko → stok beda, ada orderan yang lolos padahal barang habis.

Dengan LookUp

  • Satu daftar produk gabungan dari kedua toko, lengkap dengan filter dan pencarian.
  • Pilih banyak produk sekaligus, isi satu nilai stok atau harga.
  • LookUp yang proses ke Shopee API dan TikTok Shop API secara paralel, dengan retry otomatis kalau ada yang gagal.

Fitur

Semua yang dibutuhin buat ngurus toko multi-platform.

6 fase pertama sudah selesai dibangun — bukan sekadar rencana di roadmap.

Untuk developer

Bisa di-deploy sendiri, gratis.

LookUp itu open source. Jalankan di lokal buat development, atau deploy backend-nya langsung ke Railway.

# 1. isi kredensial Shopee + TikTok
cp .env.example apps/backend/.env

# 2. jalankan postgres + redis + backend
cd docker && docker-compose up -d

# 3. migrasi database
cd ../apps/backend && npm run db:migrate

# 4. jalankan aplikasi mobile
cd ../mobile && npm install && npm start
